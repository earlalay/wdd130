Prophet Card

The project was to design a profile card for the current prophet that included important dates and a quote. The purpose of doing this was to practice the basics of html and css. This was one of my first times using grid in css.

Apostle Spotlight

This project was to build a web page that had the photos of the current church leadership, meaning the first presidency and the twelves apostles. The point of this was continue practing html and css. I got better at using grid in css.

Favorite Devotional

The challenge was to make a table that stored information about favorite devotionals, including the speaker, title, location, and year. There is also a form to add a new devotional to the table. I learned the basics of making a table and a form in html, and then styling it in css.

Country Flags

The objective of this project was to make some country's flags by using grid in css and practice manipulating where things sit in columns and rows.

CSS Positioning

The goal of this challenge was to manipulate different colored boxes by using grid in css to form certain patterns. It was good practice by moving things in a grid with rows and columns.

Image Gallery

This assignment wanted us to practice changing the file size of an image, as well as changing the pixel size. It taught how large image files can slow the loading time of a website. It was my first introduction to using two sizes of the same image depending on what it was being used for at the time.

Signup Form

This project had us make a form that people could fill out to sign up for a daily scripture. I think it was my first time manipulating a form using grid. The whole thing presented many small, unique challenges. One was trying to get text to sit on top of an image.
